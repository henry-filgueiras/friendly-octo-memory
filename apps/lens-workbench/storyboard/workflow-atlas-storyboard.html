Annotated storyboard

Workflow Atlas turns one run journal into something you can inspect, branch, and replay.

Atlas is not a new source of truth. It is a richer operator-facing projection over the same append-only run journal: checkpoints, transforms, artifact states, transcript events, and recipe progress all stay connected.

Run projection Checkpoint anchors Recipe overlay Transcript sync Fork from any moment

How to read it

1. The journal is still the truth Atlas only projects event and artifact state; it does not invent workflow state of its own.
2. Recipe progress stays visible Completed recipe steps are marked in the graph so the manual handoff path stays legible.
3. Branching stays explicit Checkpoints and fork actions are surfaced right next to the selected event, not hidden behind automation.
